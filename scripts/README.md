# Scripts

Utility scripts for running and testing Consciousness OS.

## Available Scripts

- **start_all.sh** - Master startup script for all services
- **muselsl_retry.sh** - Automatic retry logic for muselsl streaming
- **test_integration.sh** - Integration test suite
- **test_muse_stream.py** - Muse streaming test

## Usage

```bash
# Start all services
./scripts/start_all.sh

# Test Muse streaming
python scripts/test_muse_stream.py
```
