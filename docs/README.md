# Consciousness OS Documentation

This directory contains all documentation for the Consciousness OS project, organized by component.

## Directory Structure

- **backend/** - Backend implementation docs (EEG streaming, ML analysis, artifact detection)
- **frontend/** - Frontend and integration docs (WebSocket, UI components)
- **copilot/** - AI Copilot implementation docs (conversation analyzer, models)
- **architecture/** - System architecture and technical stack docs
- **planning/** - Development planning, phase reviews, and improvement tracking
- **session-reviews/** - Session-specific reviews and progress reports
